# pinc-official Tap

## How do I install these formulae?

`brew install pinc-official/tap/<formula>`

Or `brew tap pinc-official/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "pinc-official/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
